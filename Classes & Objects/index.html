Prototypes

JS object have a special property called prototype which has special properties and methods

If object & prototype have same method, object's method will be used


Classes

Class is a program code template for creating objects.

Those objects will have some state(variables) & some behaviour(functions) inside it.

Whenever object is created in js , constructor is created which initilizes object but we can also create our own custom constructor .

Inheritance

  1. Inheritance is passing down properties & methods from parent class to child class.
  2. If child & parent have same method, child's method will be used.

Super Keyword

The super keyword is used to call the constructor of its parent class to access the parent's properties and methods.
